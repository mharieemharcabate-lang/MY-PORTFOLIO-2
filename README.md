# Automatic Plant Watering System

# Project Overview

The Automatic Plant Watering System is a smart irrigation project designed to automatically water plants when the soil becomes dry. The system uses a soil moisture sensor to monitor moisture levels and activates a water pump when needed. This project helps conserve water and ensures plants stay healthy even when the user is away.


---

# Features

Automatic watering based on soil moisture level
Uses Arduino for control and automation
Water pump activation using relay module
Simple and low-cost design
Suitable for home gardens and small plants

# How It Works

1. The soil moisture sensor checks the moisture level of the soil.
2. If the soil becomes dry, the Arduino sends a signal to the relay module.
3. The relay activates the water pump.
4. Water is supplied to the plant.
5. Once the soil reaches the desired moisture level, the pump automatically turns off.
